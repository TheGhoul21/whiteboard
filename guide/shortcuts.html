Platform-Specific ​

macOS ​

  • Cmd = Command key (⌘)
  • Option = Alt key (⌥)
  • Control = Control key (⌃)

Windows/Linux ​

  • Ctrl = Control key
  • Alt = Alt key

Note: Most shortcuts use Cmd on macOS and Ctrl on Windows/Linux.

Tips & Tricks ​

Power User Workflow ​

Left hand on keyboard:

  • H - Pan with mouse
  • P - Draw
  • E - Erase mistakes
  • T - Add labels

Right hand on mouse:

  • Draw continuously
  • Switch tools with left hand
  • Never reach for toolbar

Quick Tool Switching ​

Without shortcuts:

  1. Move mouse to toolbar
  2. Click tool
  3. Move back to canvas
  4. Continue drawing

With shortcuts:

  1. Press key (e.g., P)
  2. Immediately continue drawing
  3. 5x faster workflow

Zoom Precision ​

Smooth zoom:

  • Hold Cmd/Ctrl
  • Slowly scroll
  • Precision zoom to exact level

Quick zoom:

  • Cmd/Ctrl + + repeatedly
  • Fast zoom in
  • Cmd/Ctrl + 0 to reset

Object Positioning ​

Rough positioning:

  • Drag object to approximate location

Fine-tuning:

  • Arrow keys for 1px nudges
  • Shift+Arrow for 10px jumps
  • Perfect alignment

Learning Path ​

Beginner ​

Start with these essential shortcuts:

  1. H - Hand tool for navigation
  2. P - Pen for drawing
  3. Cmd/Ctrl + Z - Undo mistakes
  4. Space + Drag - Pan canvas
  5. Cmd/Ctrl + Scroll - Zoom

Goal: Navigate and draw without toolbar clicks.

Intermediate ​

Add these for efficiency:

  1. S - Smooth pen for better strokes
  2. E - Eraser for cleanup
  3. T - Text for labels
  4. L - Highlighter for emphasis
  5. Shift + Click - Straight lines

Goal: Complete workflow without mouse reaching toolbar.

Advanced ​

Master these for professional speed:

  1. M - Magic wand for perfect shapes
  2. Shift + P - Spotlight for presentations
  3. Cmd/Ctrl + Shift + P - Presentation mode
  4. Arrow keys - Precise object positioning
  5. F11 - Quick fullscreen toggle

Goal: Seamless recording workflow with no UI interaction.

Troubleshooting ​

Shortcut not working ​

Check:

  • Canvas is focused (click canvas first)
  • Not typing in text field
  • Correct platform modifier (Cmd vs Ctrl)
  • No conflicting browser extensions

Wrong tool activates ​

Solutions:

  • Ensure using correct key
  • Check keyboard layout (QWERTY assumed)
  • Disable browser extensions
  • Refresh page

Zoom shortcuts conflict ​

If browser zoom interferes:

Chrome/Edge:

  1. Settings → Appearance
  2. Disable browser zoom shortcuts
  3. Or use trackpad pinch instead

Firefox:

  1. about:config
  2. Modify zoom shortcut preferences

Can't remember shortcuts ​

Solutions:

  • Hover over tools for tooltip hints
  • Print cheat sheet above
  • Practice one shortcut at a time
  • Muscle memory develops quickly

Shortcut Philosophy ​

Whiteboard shortcuts follow these principles:

  1. Mnemonic: P for Pen, T for Text, E for Eraser
  2. Common patterns: Follow standard editing conventions
  3. Hand-friendly: Most tools accessible with left hand
  4. Non-modal: Switch tools anytime without mode switching
  5. Progressive: Start simple, add advanced shortcuts over time

Master shortcuts gradually - focus on your most-used tools first!
