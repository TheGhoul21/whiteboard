Features Overview ​

Whiteboard is packed with powerful features for creating engaging educational content. Explore what makes Whiteboard the perfect tool for educators.

Drawing & Annotation ​

Professional Tools ​

Pen Tool

  • Precise, solid strokes for technical drawings
  • Configurable size and color
  • Pressure sensitivity support
  • Learn more →

Smooth Pen

  • Calligraphy-style variable-width strokes
  • Natural handwriting feel
  • Speed-based stroke width
  • Perfect for expressive content

Highlighter

  • Semi-transparent emphasis
  • Overlays existing content
  • Wide strokes for visibility
  • Color-coding support

Shape Tools ​

  • Rectangles: Perfect boxes for diagrams
  • Circles: Clean circular shapes
  • Arrows: Directional indicators
  • Magic Detection: Transform rough shapes to perfect geometry

Code Blocks

  • Syntax highlighting for all major languages

Images

Animations ​

Magic Animations ​

Magic Square

  • Detect rough rectangles
  • Morph to perfect shape
  • Smooth 60fps animation
  • Learn more →

Magic Circle

  • Transform rough circles
  • Perfect circular geometry

Laser Pointer ​

  • Temporary red strokes
  • 60% initial opacity
  • 3-second smooth fade
  • Auto-removal after 15 seconds
  • Perfect for live presentations

Spotlight Effect ​

  • Real-time mouse following
  • Dims surrounding content
  • Golden glow highlight
  • Above all content (including code blocks)
  • < 5ms sync latency

Presentation Mode (Desktop App) ​

Dual-Window Architecture ​

Control Window

  • Full UI with all tools
  • For you, the presenter
  • Source of truth for all content

Presentation Window

  • Clean canvas-only view
  • For your audience
  • Perfect for OBS capture
  • Frameless with custom background

Real-Time Sync

  • < 5ms latency
  • 60fps smooth updates
  • Everything syncs: strokes, tools, spotlight, laser
