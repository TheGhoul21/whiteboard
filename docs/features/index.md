# Features Overview

Whiteboard is packed with powerful features for creating engaging educational content. Explore what makes Whiteboard the perfect tool for educators.

## Drawing & Annotation

### Professional Tools

**Pen Tool**
- Precise, solid strokes for technical drawings
- Configurable size and color
- Pressure sensitivity support
- [Learn more →](/features/drawing-tools)

**Smooth Pen**
- Calligraphy-style variable-width strokes
- Natural handwriting feel
- Speed-based stroke width
- Perfect for expressive content

**Highlighter**
- Semi-transparent emphasis
- Overlays existing content
- Wide strokes for visibility
- Color-coding support

### Shape Tools

- **Rectangles**: Perfect boxes for diagrams
- **Circles**: Clean circular shapes
- **Arrows**: Directional indicators
- **Magic Detection**: Transform rough shapes to perfect geometry

[Explore Drawing Tools →](/features/drawing-tools)

## Smart Objects

### Rich Content Support

**LaTeX Math**
- Beautiful equation rendering
- Full KaTeX support
- Inline and block equations
- Editable after creation
- [Learn more →](/features/latex)

**Code Blocks**
- Syntax highlighting for all major languages
- CodeMirror editor integration
- Auto-detection from paste
- Line numbers and themes
- [Learn more →](/features/code-blocks)

**Text Objects**
- Editable text labels
- Custom fonts and sizes
- Movable and resizable
- Perfect for annotations

**Images**
- Import via paste or drag-and-drop
- Maintain quality at any zoom
- Resizable with aspect ratio lock
- Perfect for reference material

## Animations

### Magic Animations

**Magic Square**
- Detect rough rectangles
- Morph to perfect shape
- Smooth 60fps animation
- [Learn more →](/features/animations)

**Magic Circle**
- Transform rough circles
- Perfect circular geometry
- Engaging visual effect

### Laser Pointer

- Temporary red strokes
- 60% initial opacity
- 3-second smooth fade
- Auto-removal after 15 seconds
- Perfect for live presentations

### Spotlight Effect

- Real-time mouse following
- Dims surrounding content
- Golden glow highlight
- Above all content (including code blocks)
- < 5ms sync latency

[Master Animations →](/features/animations)

## Frame-Based Navigation

### Multi-Page Canvas

**Frames**
- Organize content into pages
- Smooth transitions between frames
- Different backgrounds per frame
- Perfect for step-by-step lessons
- [Learn more →](/features/frames)

**Navigation**
- Click frame to navigate
- Smooth pan/zoom transitions
- Keyboard shortcuts
- Frame thumbnails

## Presentation Mode (Desktop App)

### Dual-Window Architecture

**Control Window**
- Full UI with all tools
- For you, the presenter
- Source of truth for all content

**Presentation Window**
- Clean canvas-only view
- For your audience
- Perfect for OBS capture
- Frameless with custom background

**Real-Time Sync**
- < 5ms latency
- 60fps smooth updates
- Everything syncs: strokes, tools, spotlight, laser

[Set up Presentation Mode →](/desktop/presentation-mode)

### OBS Integration

**Recording Features**
- Frameless window capture
- Custom background colors
- Chroma key support (green screen)
- Always-on-top option
- Fullscreen toggle

[Configure OBS →](/desktop/obs-integration)

## Canvas Features

### Infinite Workspace

- Unlimited canvas size
- Smooth pan and zoom
- Multiple zoom levels (10%-500%)
- Keyboard and mouse navigation

### Backgrounds

Choose from multiple background styles:
- **None**: Clean white/transparent
- **Grid**: Dot grid for alignment
- **Lines**: Notebook-style ruled lines
- **Dark**: Dark mode for contrast

### Auto-Save

- Automatic saving every 10 seconds
- localStorage persistence
- No manual save needed
- Work preserved across sessions
